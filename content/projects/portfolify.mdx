## <a href="https://portfolify-project.netlify.app/" target="_blank" rel="noopener noreferrer"> Live Demo</a>

This is a dynamic portfolio management application that enables users to create, customize, 
and publish their personal portfolios with various templates. 
Built with React (Vite) and styled with Tailwind CSS, 
this front end interacts with the backend via REST API and integrates Cloudinary for media uploads.

## Features

- **User authentication**: Login, Register, JWT-based Auth.
- **Create & manage portfolios**: CRUD operations.
- **Slug Generation**: Portfolify automatically generates a unique URL (slug) for each portfolio, making sharing their portfolio with potential employers or clients easier.
- **Responsive Design**: The website is responsive and works on all devices.

## Technologies

- **React (Vite)**: Fast & optimized frontend framework.
- **Axios**: API requests.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **Netlify**: A cloud platform for static sites and serverless functions.

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone https://github.com/your-username/portfolify-client.git
cd portfolify-client
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.

## Deployment

You can manually upload the `/dist` folder to Netlify or connect your GitHub repository for automatic deployment.

Portfolify is a modern and flexible portfolio management solution designed to provide users with an intuitive way to create, customize, and publish their personal portfolios.
